# Building

## Building on Linux (and maybe others)

1. Clone the repo.
2. `cd simple_harness`
3. `mkdir build`
4. `cd build`
5. `cmake ..`
6. `cmake --build .`
7. `cd ..`
8. `./build/simple_harness`

## Building on MacOS
1. Clone the repo.
2. `cd simple_harness`
3. `gcc -o compile compile.c`
4. `./compile`
5. `./simple_harness`

## Building on Windows
0. Install (w64devkit)[https://github.com/skeeto/w64devkit]
1. Follow instructions for Building on Linux.
